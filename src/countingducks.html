Counting Ducks
ios/ android
My first do-it-yourself mobile game! It’s simple, count the ducks that are in the house when the timer stops. It’s fun and frustratingly hard, but some technical constraints like being able to save the game or pause the game add some unwanted frustration. I’ve since then learned how to save and pause an app game and that’s what this project was all about. Learning. I learned so much about game development and marketing, also was incredible fun getting this one on the apple and android stores. Proud to say ALL coding and ALL artwork was done by yours truly.
